--Plasma is a Proof of concept which allows Bootstrap 4 and its animations, as well as
Font Awesome and its iconography to be fully utilized in a website / web app
without an Internet Connection or use of a CDN.
The Bootstrap folder provided is slightly arranged with src files from Jquery, popper and Font Awesome
to allow the functionalities stated in the Plasma Project to be implimented.



--Bootstrap 4 is a front-end development framework that is used to create
professional website front-ends while minimizing workload on developers.

Font Awesome is a compilation of Icons that are used to generalize website
interfaces with icons.
